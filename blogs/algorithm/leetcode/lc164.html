164. 最大间距

gouqi 2020-11-26 leetcode线性排序

# 题目

# 题目分析

  这个题，线性时间复杂度排序，开始还想着用什么单调栈之类的看看能不能绕过排序，后来发现不行，那就用之前介绍过的三种排序方式，基数排序，计数排序，桶排序。
  其中计数排序不行，因为计数排序需要有一个从最大值到最小值的遍历，而如果题目只有两个数，但一个是 INT_MIN,一个是 INT_MAX,那也需要从 INT_MIN 到 INT_MAX 的遍历，所以不满足线性的要求。
  基数排序可以，就是按照基数排序的思路每一层去对“位”排序，知道所有位都排序即可，注意编程的一些技巧就行。

# 基数排序代码

    int maximumGap(vector<int>& nums) {
        int n=nums.size();
        if (n < 2) {
            return 0;
        }
        int maxValue = *max_element(nums.begin(),nums.end());
        int exp = 1;
        vector<int>buf(n);
        while (maxValue >= exp) {
            vector<int>cnt(10);
            for (int i = 0; i < n; i++) {
                int cur = (nums[i] / exp) % 10;
                cnt[cur]++;
            }
            for (int i = 1; i < 10; i++) {
                cnt[i] += cnt[i-1];
            }
            for (int i = n-1; i >= 0; i--) {
                int cur = (nums[i] / exp) % 10;
                buf[cnt[cur] - 1] = nums[i];
                cnt[cur]--;
            }
            copy(buf.begin(),buf.end(),nums.begin());
            exp *= 10;
        }
        int ans = 0;
        for (int i = 1; i < n; i++) {
            ans = max(ans,nums[i] - nums[i-1]);
        }
        return ans;
    }

  桶排序，其实这里不能完全叫做桶排序，就是用桶排序的思想，先分桶，但是呢不用对桶内元素真正进行排序，只需要保存桶内元素的最大值和最小值即可，因为各个桶之间是有序的，所以相邻后一个桶中的最小值一定和上一个桶中的最大值相邻，所以用这个的差值可以找出相邻元素的最大值。当然这里能这样用也是分桶上面做了一些小技巧。
如何划分桶?
  这里是根据数据范围来划分桶的，我们设定桶的数据范围为 d = (maxValue - minValue) / (n - 1)，n 为数据个数，就相当于在 minValue 到 maxValue 中平分了 n-1 段，同时桶的个数为 bucketSize = (maxValue - minValue)/d +1。说起来可能比较抽象，举个栗子，数据为[1,3,11]，那么 d=(11-1)/2=5,这时候桶的个数 bucketSize = (11-1)/5 +1=3，就是有三个桶，其依次装的数据范围为[1-5],[6-10],[11-15]。仙子啊看懂桶的个数哪里为什么要加一了吗，就是防止数据不能完全整除的情况。
为什么桶内相邻元素的大小不用考虑？
  上面说了分桶的方法，我们可以根据分桶的方法来想一下，其每个桶的数据范围相当于平均了（maxValue-minValue）。打个比方，一条公路上有 4 个路标，其中首尾各一个，问这四个路标中相邻路标的最大距离是多少，那我们可以想到当四个路标平均分布的时候是不是这时候相邻路标的最大距离距离最小，就是总的距离除以 3。因为如果不平均，中间两个路标无论怎么放，都会有一个距离比平均分布的时候大.(你离上一个路标近了就离下一个路标远了...)。

# 桶排序代码

    int maximumGap(vector<int>& nums) {
        int n=nums.size();
        if (n < 2) {
            return 0;
        }
        int maxValue = *max_element(nums.begin(),nums.end());
        int minValue = *min_element(nums.begin(),nums.end());
        int d = max(1, (maxValue - minValue) / (n - 1));  // 题目说了为非负整数，但可能存在0的情况，1，1，2.
        int bucketSize  = (maxValue - minValue) / d + 1; // 向上取整
        vector<pair<int,int> >bucket(bucketSize,{-1,-1});  // -1, -1 表示桶为空,first表示最大值，second表示最小值
        for (int i = 0; i < n; i++) {
            int index = (nums[i] - minValue) / d;
            if (bucket[index].first == -1) {
                bucket[index].first = nums[i];
                 bucket[index].second = nums[i];
            }
            else{
                bucket[index].first = max(bucket[index].first,nums[i]);
                bucket[index].second = min(bucket[index].second,nums[i]);
            }
        }
        int ans = 0;
        int preIdx = -1;
        for (int i = 0; i < bucketSize; i++) {
            if (bucket[i].first != -1) {
                if (preIdx != -1) {
                    ans =max(ans, bucket[i].second - bucket[preIdx].first);
                }
                preIdx = i;
            }
        }
        return ans;
    }
